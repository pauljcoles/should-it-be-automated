# Product Overview

Test Prioritisation Scoring Tool - A React-based application for evaluating test automation candidates using Angie Jones' risk-based scoring methodology.

## Purpose

Helps QA teams make objective decisions about test automation by calculating priority scores based on multiple risk factors. Replaces gut-feel decisions with data-driven recommendations.

## Key Features

- **Objective Scoring**: Calculate automation priority based on risk, value, ease, history, and legal factors
- **Two Modes**: 
  - Normal mode: Simple calculator using Angie Jones' exact model (0-80 scale)
  - Teaching mode: Calculator with educational guidance for learning teams (0-100 scale)
- **Visual Recommendations**: Color-coded guidance (AUTOMATE/MAYBE/DON'T AUTOMATE)
- **State Diagram Integration**: Import model-based testing diagrams to auto-generate test cases
- **Data Portability**: Export/import as JSON for team collaboration
- **Client-Side Only**: No backend required, all data stored in browser localStorage

## Target Users

QA teams, test automation engineers, and development teams making decisions about test automation priorities.
